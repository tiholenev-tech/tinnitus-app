## 1. 🔴 АУДИО БЪГ — ДИАГНОЗА ПОТВЪРДЕНА (02.06 сутрин)

### Симптом и тестове:
- 5 нощи (вкл. снощи) работи БЕЗ проблем. Тази сутрин аудиото спря само̀.
- ТЕСТ ПОТВЪРДИ: на **отключен екран** върви >1 час безпроблемно. На **ЗАКЛЮЧЕН екран спира за ~5 мин.**
- При отключване app се "обновява сам" (като refresh) → връща се на начална страница.

### Диагноза (категорична):
НЕ е конкретен файл, НЕ е логически бъг в кода. Причината е **OS power management при заключен екран** — Android suspend-ва/убива PWA таба за пестене на батерия. Wake Lock e 'screen' тип → умира при заключване → OS suspend → аудио спира → при отключване app се презарежда от нула.

### Защо PWA не може да го реши напълно:
PWA няма native foreground service / audio focus, който да каже на OS "не ме убивай, свиря". Това е същинското PWA ограничение.

### Решения (ред):
1. **Временна кръпка (PWA):** Media Session API (navigator.mediaSession, setActionHandler play/pause, playbackState='playing'). Android често третира media session като foreground media и НЕ suspend-ва. Не гарантирано, но си струва преди Capacitor.
2. **Истинско решение: Capacitor** (foreground service / native audio) — раздел 3.
3. Прочети logs/audio.log (телеметрията вече трябва да е хванала случая — търси wakelock_released + повторен session_start = презареждане).

## 2. СТРАТЕГИЧЕСКИ РЕШЕНИЯ

### Дистрибуция: web-базиран продукт + Capacitor обвивка
- Сайтът (tinnitus-app.help) е сърцето: landing + статии + тест + плащане. SEO/GEO трафик.
- Capacitor = server-served APK (тънка обвивка зареждаща живия сайт). Ъпдейтваш сайта → app нов без нов APK.
- Capacitor НЕ е в конфликт с web стратегията — опакова същия сайт + добавя native слой.

### Capacitor решава ТРИ неща (затова е обоснован):
1. Заключен екран (днешният бъг) — foreground service държи аудиото
2. Audio focus при обаждане — паузира при звънене, връща след разговора
3. Дистрибуция (Google Play или директен APK)

### Пазари: България ПЪРВО → Италия ПОСЛЕ
- BG първа: роден език, бързо/евтино тестване, гладене на бъгове, валидиране. НЕ за реклама-печалба (ниски доходи).
- Италия = приходен пазар след изпипване на BG + качествен превод. Най-застаряваща ЕС (24.7% над 65), частни здравни разходи, слаба конкуренция на "acufeni".
- Платежоспособност > заболеваемост: BG/RO/GR имат най-тежък тинитус но ниски доходи (не плащат). Италия/Нидерландия/Франция = пари. Германия + английски ИЗБЯГВАЙ.

### Цена: €19.99 еднократно + 14-дневен trial
- НЕ абонамент. Плащаш веднъж, твое завинаги.
- Променливи отстъпки според конверсията (управляемо от admin).
- Оптимизирай по ПРИХОД НА ПОСЕТИТЕЛ, не по сваляния. Не на малък трафик.
- Малки FB реклами с отстъпки = допълнителен канал към SEO.

### Безплатно vs платено:
- **Landing = напълно безплатен:** статии (SEO) + тест + аха-момент + **1-2 звука се чуват на самата страница** ПРЕДИ сваляне (убеждава "работи").
- **App:** тест + облекчение задължително вътре; пълен 14-дневен trial → €19.99 отключва библиотека (~256 звука), миксер, дневник ЗАВИНАГИ.
