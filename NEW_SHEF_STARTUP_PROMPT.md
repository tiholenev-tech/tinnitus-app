- Дай handoff документ преди да изтощиш context

### Известни pitfalls
- Cloudflare quick tunnels са flaky — често падат

- 4-ти Code актьор = chaos, file ownership конфликт. Максимум 3.
- Опус research документите имат конкретни числа — никога не measure-вай без тях
