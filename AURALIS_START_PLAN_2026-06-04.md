## КЛЮЧОВО РЕШЕНИЕ: WEB-FIRST, Capacitor НА ИЗЧАКВАНЕ

### Защо web-first (потвърдено днес 04.06):
- Google Play взема 15-30% (€3-6 на €19.99) + изисква плащането през тях (не Stripe вътре). Твърде скъпо.
- PWA + Stripe = 0% комисиона, всичко твое.
- PWA + cloud sync покрива ВСИЧКО: плащане, данни, смяна на телефон, дистрибуция.
- Единственото, което PWA НЕ решава = аудио при заключен екран/обаждане.

### Capacitor се свежда до ЕДИН въпрос: аудио при заключен екран
- За app за СЪН това е същината (звукът трябва да върви цяла нощ при заключен телефон).
- ПЛАН: пробвай Media Session API ПЪРВО (евтина PWA кръпка). Android често третира media session като foreground media → не suspend-ва.
  - Ако Media Session реши заключения екран → чист PWA, БЕЗ Capacitor, пускаш бързо.
  - Ако НЕ → Capacitor APK директно от сайта (НЕ Play, за да пазиш 0% комисиона). Цена: "непознат източник" бариера за 50+ (смекчава се с инструкции).
- Capacitor НЕ е за: смяна на телефон (това е cloud sync), плащане (Stripe), дистрибуция (PWA/директен APK).

## ДВЕ НИВА НА ПРОДУКТА (важно продуктово решение)

### Landing сайт = ОПРОСТЕНА кука
- Студен посетител от Google/реклама — НЕ бива сложния миксер веднага (обърква, отказва).
- Минимална, почти автоматична версия: направи теста → чуй облекчението (аха-момент) → 1 готов автоматично настроен звук → "уау, работи".
- 1-2 звука се чуват на самата страница ПРЕДИ сваляне/плащане.
- Простотата ПРОДАВА тук, не богатството на функции.

### App = пълният продукт (за платилите)
- Богат миксер, ~256 звука, дневник, програма, всичко.

## ПЛАН ЗА СТАРТ — ПОДРЕДЕН ЧЕКЛИСТ

### СЕДМИЦА 1 — техническа основа
1. **Media Session кръпка** → тествай заключен екран. Реши Capacitor да/не.
2. **Cloud sync + login (magic link)** — данни на сървъра, смяна на телефон, възстановяване на покупка. MySQL: profiles/favorites/pitch/thi/diary/streak. ≈ €0 (язди на droplet-а; само email услуга единици €).
3. **Stripe checkout** (регистрацията ИМА) — €19.99 еднократно, webhooks, промо/отстъпки от admin.
4. **изи пей** — за БГ пазара (местните го предпочитат).
5. **Anti-fraud** — server-side: 1 email=1 trial (MySQL) + magic link verify + IP лимит (3/IP/30дни). НЕ cookie-based.

### СЕДМИЦА 1-2 — landing + правни
6. **Landing сайт (PHP, Bichromatic)** — опростена кука (тест + 1 звук) + 1-2 звука на страницата + paywall. SEO/GEO от ден 1 (виж AURALIS_MASTER).
7. **Support email** — реален адрес + някой да отговаря.
8. **Refund политика + възстановяване на покупка** (Stripe изисква).
9. **Условия + Privacy + GDPR → български юрист** (чернова готова, чака преглед).

### ПАРАЛЕЛНО — съдържание (за SEO трафик)

## МАРКЕТИНГ — какво бяхме решили (за да не забравиш)

### Funnel:
FB видео реклама (mute-friendly, субтитри — reels се гледат без звук) → клик → landing → опростена кука (тест + звук + аха-момент) → email → paywall (€19.99).

### Канали:
1. **SEO/GEO органичен трафик** (ОСНОВНИЯТ канал) — статии на български, после италиански. Хората търсят "шум в ушите" / "tiuit in urechi" в Google → намират статия → влизат.
2. **Малки FB реклами** с отстъпки (ДОПЪЛНИТЕЛЕН) — големи субтитри, отстъпки, за 50+.

### Пазари:
- БЪЛГАРИЯ ПЪРВО (тест, роден език, евтино, гладене на бъгове).
- ИТАЛИЯ ПОСЛЕ (приходен пазар — застаряващи, платежоспособни, слаба конкуренция на "acufeni"). DeepL превод + УНГ верификация.
- Избягвай Германия/английски (свръхконкуренция).

### Важни ограничения:
- Health claims във FB ads ограничени → само wellness формулировки ("управление", "облекчение на възприятието", НЕ "лекува").
- YMYL/E-E-A-T: статиите трябват реален УНГ рецензент + научни линкове, иначе Google не ги ранкира (67%+ неоптимизирани загубиха видимост).
- robots.txt: блокирай обучаващите ботове, разреши търсещите (детайли в AURALIS_MASTER).

### Цена-стратегия:
€19.99 базова, променливи отстъпки според конверсията. Оптимизирай по ПРИХОД НА ПОСЕТИТЕЛ, не по брой сваляния. Не оптимизирай на малък трафик.
